# Electronics Competition Project

## Environment Setup

### Virtual Environment
- Location: `./venv`
- Python version: 3.13

### Activation
```bash
source venv/bin/activate
```

### Installed Dependencies
- OpenCV (headless) 5.0.0
- NumPy 2.5.1
- Flask 3.1.3

## Directory Structure
- `src/` - Source code
- `tests/` - Test files
- `docs/` - Documentation
- `samples/` - Sample data and images
- `calibration_data/` - Camera calibration data

## Usage
1. Activate the virtual environment: `source venv/bin/activate`
2. Run your Python scripts from the project directory
3. Use `deactivate` to exit the virtual environment
